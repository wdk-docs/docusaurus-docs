# 📦 theme-mermaid

这个主题提供了一个由[Mermaid](https://mermaid-js.github.io/)提供支持的`@theme/Mermaid`组件。您可以在[图表](../../guides/markdown-features/markdown-features-diagrams.mdx)文档中阅读更多信息。

```bash npm2yarn
npm install --save @docusaurus/theme-mermaid
```

## 配置 {#configuration}

```js title="docusaurus.config.js"
module.exports = {
  themes: ["@docusaurus/theme-mermaid"],
  // In order for Mermaid code blocks in Markdown to work,
  // you also need to enable the Remark plugin with this option
  markdown: {
    mermaid: true,
  },
};
```
