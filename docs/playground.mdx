# Playgrounds

Playgrounds 允许您**在浏览器中运行 Docusaurus，而无需安装任何**!

它们主要用于:

- 测试 Docusaurus
- 报告错误

使用[docusaurus.new](https://docusaurus.new)作为易于记忆的快捷方式。

从下面的可用选项中选择一个。

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

为方便起见，下次您访问[docusaurus.new](https://docusaurus.new)时，我们会记住您的选择。

:::
