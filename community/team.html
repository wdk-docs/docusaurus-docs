团队

活跃的团队​

Docusaurus 团队致力于核心功能，经典主题的插件，以及 Docusaurus 文档网站。

下面按字母顺序列出了现在的 Docusaurus 团队成员。

Alexey Pyltsyn

痴迷的开源爱好者👋永远的业余爱好者在一切🤷‍♂️俄语文档的维护者在PHP, React, Kubernetes和更多🧐
GitHub

Joel Marcey

Docusaurus的创始人，现在一直感激Docusaurus的啦啦队长，感谢那些为它写代码的人。
GitHubTwitter

Sébastien Lorber

自2014年以来的反应恋人。自由撰稿人，帮助Facebook发布Docusaurus v2。他定期在他的{网站}和Dev.to上写作。
GitHubTwitter

Sida Chen

来自中国上海的学生。热心的开源项目创建者，但从未真正在他创建的项目上努力工作。
GitHubTwitter

Yangshun Tay

喜欢在Jamstack上工作的全前端堆栈开发人员。在《Docusaurus》上的工作让他成为Facebook非官方的兼职开源网站管理员，这是一个很棒的角色。
GitHubTwitter

荣誉校友​

如果没有这些人的巨大贡献，Docusaurus 永远不会有今天的样子，他们已经转向了更大更伟大的东西。

Endilie Yacop Sucipto

Maintainer @docusaurus · 🔥🔥🔥
GitHubTwitter

Wei Gao

🏻‍🌾 正在进行的React开发人员，维护Docusaurus，编写文档并在许多网站上发送垃圾邮件。
GitHubTwitter

学生们​

少数学生也曾在 Docusaurus 上工作过，这是他们学校学期/实习和大联盟黑客奖学金计划的一部分，他们贡献了一些令人惊叹的特性，比如插件选项验证、迁移工具和 Bootstrap 主题。

Anshul Goyal

热爱编码和尝试新技术的全栈开发人员。在空闲时间，他为开源做贡献，在他的website上写博客文章，看动漫。
GitHubTwitter

Drew Alexander

开发人员和创意人员，试图获得技能来构建他能想到的任何东西。
GitHub

Fanny Vieira

范妮在高中时就开始了网络开发，为学校厨房建立了一个项目。在业余时间，她喜欢为开源做贡献，偶尔在她的博客上写一些关于她的经历、烹饪和创建Spotify playlists的文章。
GitHubTwitter

Sam Zhou

Sam于2011年开始编程，并于2015年建立了自己的websiteLink。他对编程语言、基础设施开发和web开发很感兴趣，并建立了自己的编程语言和迷你 React。
GitHubTwitter

Tan Teik Jun

开源爱好者的目标是成为像这个页面上的其他人一样令人敬畏的人。研究Docusaurus让他离目标更近了一步。🌱
GitHubTwitter

Nisarag Bhatt

热爱学习和应用新技术的全栈web开发人员!喜欢为开源做贡献，也喜欢写内容文章和教程。
GitHubTwitter
