很棒的资源

一个有趣的 Docusaurus 社区项目列表。

社区文档​

Docusaurus.community站点扩展了官方文档，提供了更复杂的示例和完整的文章，这些文章推荐了最佳实践并提供了常见问题的解决方案。

它也是一个完整的 Docusaurus 应用程序，就像我们的主站点一样，因此定制和实验示例非常适合托管在那里。

信息

我们可能会允许在我们的官方文档中添加指向社区文档的直接链接，如果社区发现这种指向文章的链接有用的话。

我们还在考虑在将来的某个时候将其进一步集成到官方文档中。

社区插件​

搜索​

  • docusaurus-plugin-lunr - Docusaurus v2 插件创建本地搜索索引，用于 Lunr.js
  • docusaurus-lunr-search - 离线搜索 Docusaurus v2
  • docusaurus-search-local - 离线/本地搜索 Docusaurus v2
  • @easyops-cn/docusaurus-search-local - 离线/本地搜索 Docusaurus v2(支持 zh 语言)
  • docusaurus-theme-search-typesense - Docusaurus v2 插件Typesense DocSearch
  • @markprompt/docusaurus-theme-search - Docusaurus 插件添加生成 AI/LLM 驱动的文档搜索，由Markprompt提供支持。

集成​

  • docusaurus2-dotenv - Docusaurus 2 插件，支持 dotenv 和其他环境变量
  • posthog-docusaurus - 整合PostHog产品分析与 Docusaurus v2
  • docusaurus-plugin-moesif - 增加了Moesif API Analytics来跟踪用户行为并精确定位开发者在你的激活漏斗中的下降位置。
  • docusaurus-plugin-yandex-metrica - 增加Yandex Metrika计数器，用于评估网站流量和分析用户行为。

特性​

  • docusaurus-theme-github-codeblock. Docusaurus v2 插件，支持从公共 GitHub 存储库引用代码示例
  • docs-to-pdf - 生成 PDF 格式的文档，适用于 Docusaurus v1 和 v2(包括测试版)
  • docusaurus-plugin-sass - Sass/SCSS 样式表支持
  • docusaurus-plugin-remote-content - Docusaurus v2 插件，允许您从远程源获取内容
  • @graphql-markdown/docusaurus - 一个 Docusaurus v2 插件，用于从 GraphQL 模式生成文档
  • docusaurus-plugin-react-docgen-typescript - Docusaurus v2 插件，用于从react-docgen-typescript加载类型注解
  • docusaurus-plugin-react-docgen - Docusaurus v2 插件，用于从react-docgen加载类型注释
  • docusaurus-plugin-module-alias - Docusaurus v2 插件，用于快速混叠本地模块
  • docusaurus-protobuffet - 用于 Protobuf 合同文档的 Docusaurus 工具集
  • docusaurus-prince-pdf - 使用 prinexml 生成 PDF，以获得更好的字体子集和 ToC 特性。支持 Docusaurus v2 站点
  • redocusaurus - 一个 Docusaurus 预置，用于通过Redoc将 OpenAPI 文档集成到您的文档中
  • plugin-image-zoom - Docusaurus 2 的图像缩放插件
  • docusaurus-plugin-typedoc - 一个 Docusaurus 2 插件，用于使用TypeDoc构建文档
  • docusaurus-openapi-docs - Docusaurus v2 插件和主题，用于生成交互式 OpenAPI 文档
  • docusaurus-post-generator - 一个命令行工具，用户可以快速添加博客/文档文件的命令，如yarn gen-post new [template] [post_name]。
