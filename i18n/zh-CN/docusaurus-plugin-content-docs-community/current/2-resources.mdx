# 很棒的资源

一个有趣的 Docusaurus 社区项目列表。

## 社区文档 {#community-documentation}

**[Docusaurus.community](https://docusaurus.community)**站点扩展了[官方文档](/docs)，提供了更复杂的示例和完整的文章，这些文章推荐了最佳实践并提供了常见问题的解决方案。

它也是一个完整的 Docusaurus 应用程序，就像我们的主站点一样，因此定制和实验示例非常适合托管在那里。

:::info

我们可能会允许在我们的官方文档中添加指向社区文档的直接链接，如果社区发现这种指向文章的链接有用的话。

我们还在考虑在将来的某个时候将其进一步集成到官方文档中。

:::

## 社区插件 {#community-plugins}

### 搜索 {#search}

- [docusaurus-plugin-lunr](https://github.com/daldridge/docusaurus-plugin-lunr) - Docusaurus v2 插件创建本地搜索索引，用于 Lunr.js
- [docusaurus-lunr-search](https://github.com/lelouch77/docusaurus-lunr-search) - 离线搜索 Docusaurus v2
- [docusaurus-search-local](https://github.com/cmfcmf/docusaurus-search-local) - 离线/本地搜索 Docusaurus v2
- [@easyops-cn/docusaurus-search-local](https://github.com/easyops-cn/docusaurus-search-local) - 离线/本地搜索 Docusaurus v2(支持 zh 语言)
- [docusaurus-theme-search-typesense](https://github.com/typesense/docusaurus-theme-search-typesense) - Docusaurus v2 插件[Typesense DocSearch](https://typesense.org/docs/latest/guide/docsearch.html)
- [@markprompt/docusaurus-theme-search](https://github.com/motifland/markprompt-js/tree/main/packages/docusaurus-theme-search) - Docusaurus 插件添加生成 AI/LLM 驱动的文档搜索，由[Markprompt](https://markprompt.com)提供支持。

### 集成 {#integrations}

- [docusaurus2-dotenv](https://github.com/jonnynabors/docusaurus2-dotenv) - Docusaurus 2 插件，支持 dotenv 和其他环境变量
- [posthog-docusaurus](https://github.com/PostHog/posthog-docusaurus) - 整合[PostHog](https://posthog.com/)产品分析与 Docusaurus v2
- [docusaurus-plugin-moesif](https://github.com/Moesif/docusaurus-plugin-moesif) - 增加了[Moesif API Analytics](https://www.moesif.com/)来跟踪用户行为并精确定位开发者在你的激活漏斗中的下降位置。
- [docusaurus-plugin-yandex-metrica](https://github.com/sgromkov/docusaurus-plugin-yandex-metrica) - 增加[Yandex Metrika](https://metrika.yandex.ru/)计数器，用于评估网站流量和分析用户行为。

### 特性 {#features}

- [docusaurus-theme-github-codeblock](https://github.com/saucelabs/docusaurus-theme-github-codeblock). Docusaurus v2 插件，支持从公共 GitHub 存储库引用代码示例
- [docs-to-pdf](https://github.com/jean-humann/docs-to-pdf) - 生成 PDF 格式的文档，适用于 Docusaurus v1 和 v2(包括测试版)
- [docusaurus-plugin-sass](https://github.com/rlamana/docusaurus-plugin-sass) - Sass/SCSS 样式表支持
- [docusaurus-plugin-remote-content](https://github.com/rdilweb/docusaurus-plugin-remote-content) - Docusaurus v2 插件，允许您从远程源获取内容
- [@graphql-markdown/docusaurus](https://github.com/graphql-markdown/graphql-markdown) - 一个 Docusaurus v2 插件，用于从 GraphQL 模式生成文档
- [docusaurus-plugin-react-docgen-typescript](https://github.com/atomicpages/docusaurus-plugin-react-docgen-typescript) - Docusaurus v2 插件，用于从`react-docgen-typescript`加载类型注解
- [docusaurus-plugin-react-docgen](https://github.com/atomicpages/docusaurus-plugin-react-docgen) - Docusaurus v2 插件，用于从`react-docgen`加载类型注释
- [docusaurus-plugin-module-alias](https://github.com/atomicpages/docusaurus-plugin-module-alias) - Docusaurus v2 插件，用于快速混叠本地模块
- [docusaurus-protobuffet](https://github.com/protobuffet/docusaurus-protobuffet) - 用于 Protobuf 合同文档的 Docusaurus 工具集
- [docusaurus-prince-pdf](https://github.com/signcl/docusaurus-prince-pdf) - 使用 prinexml 生成 PDF，以获得更好的字体子集和 ToC 特性。支持 Docusaurus v2 站点
- [redocusaurus](https://github.com/rohit-gohri/redocusaurus) - 一个 Docusaurus 预置，用于通过[Redoc](https://github.com/redocly/redoc)将 OpenAPI 文档集成到您的文档中
- [plugin-image-zoom](https://github.com/flexanalytics/plugin-image-zoom) - Docusaurus 2 的图像缩放插件
- [docusaurus-plugin-typedoc](https://github.com/tgreyuk/typedoc-plugin-markdown/tree/master/packages/docusaurus-plugin-typedoc) - 一个 Docusaurus 2 插件，用于使用[TypeDoc](https://typedoc.org/)构建文档
- [docusaurus-openapi-docs](https://github.com/PaloAltoNetworks/docusaurus-openapi-docs) - Docusaurus v2 插件和主题，用于生成交互式 OpenAPI 文档
- [docusaurus-post-generator](https://github.com/moojing/docusaurus-post-generator) - 一个命令行工具，用户可以快速添加博客/文档文件的命令，如`yarn gen-post new [template] [post_name]`。
