# 📦 theme-search-algolia

这个主题提供了一个`@theme/SearchBar`组件，可以轻松地与 Algolia DocSearch 集成。结合`@docusaurus/theme-classic`，它提供了一个非常简单的搜索集成。您可以在[search](../../search.mdx)文档中阅读更多信息。

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

这个主题还添加了搜索页面可用在`/search`(作为 swizzlable `SearchPage`组件)路径与 OpenSearch 支持。您可以通过`themeConfig.algolia.searchPagePath`更改此默认路径。使用`false`来禁用搜索页面。

:::tip

如果你已经安装了`@docusaurus/preset-classic`，你不需要将其作为依赖项安装。

:::
