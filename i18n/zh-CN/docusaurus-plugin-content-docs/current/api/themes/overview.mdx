# Docusaurus 主题

我们提供官方的 Docusaurus 主题。

## 主题 {#main-themes}

主要主题实现了[docs](../plugins/plugin-content-docs.mdx)、[blog](../plugins/plugin-content-blog.mdx)和[pages](../plugins/plugin-content-pages.mdx)插件的用户界面。

- [@docusaurus/theme-classic](./theme-classic.mdx)
- 🚧 计划其他主题

:::caution

目标是让所有主题共享完全相同的功能、用户体验和配置。

只有 UI 设计和底层样式框架应该改变，并且您应该能够轻松地更改主题。

我们还没有做到这一点:只有经典主题才可以投入生产。

:::
